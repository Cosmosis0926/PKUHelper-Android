PKU Helper-Android
================

### 使用说明
  1. 安装好Android Studio http://pan.baidu.com/s/1i3jgELR
  2. 下载此项目源代码，并导入项目（注意项目编码已经是UTF-8了）
  3. 修改app/build.gradle中的signingConfigs项，将其debug.keystore设置为你下载后的debug.keystore文件路径
  4. 点击RUN运行吧！
